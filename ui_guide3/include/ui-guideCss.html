에디터 설정

CSS 파일 최상단에 언어셋을 지정한다. (신규 페이지 인코딩 선언은 utf-8를 원칙으로 한다.

외부 스타일시트(<link />)를 사용한다. (@import는 일부 브라우저에서 HTML 문서 로딩 후에야 로드되므로 사용하지 않는다)

소프트 탭 설정은 공백 문자 4개로 합니다.

파일의 맨 마지막은 줄바꿈으로 끝나게 합니다.

default.css

스타일 규칙

클래스 네이밍

첫 시작에 숫자, 대문자, 특수문자는 사용하지 않는다.

두 단어의 조합은 언더바(_)를 사용한다.

시각적 표현보다는 요소의 구조, 목적이 표현되도록 한다. 형태_의미_상태로 조합하며 3단계를 넘어가지 않도록 작명한다.

기본 문법

스타일은 한 줄에 작성하며, 줄바꿈 시 빈 줄이 생기지 않게 한다. 스타일 속성 간 개행하지 않는다.

클래스명을 선언한 뒤 한 칸의 공백을 두고, 세부 속성간에는 한 칸의 공백을 준다. 속성과 값 사이는 띄어쓰지 않는다. 선택자 {속성:값; 속성:값;}

CSS 선택자는 class로 핸들링되며, 태그네임으로 핸들링하지 않는다. (단 사용빈도가 높은 li, td, th는 예외) 또한, 태그네임과 class는 중복사용하지 않는다.

속성 선언에 따옴표는 사용하지 않는다. (단, 한글폰트 선언&공백표현 등 사용이 필요한 경우는 예외이며, 사용 시에는 홑따옴표로 통일한다.)

background-position 속성값은 숫자로 선언한다.

컬러값은 축약형을 사용한다.

z-index 수치 간격은 10단위로 적용한다.

속성 선언 순서

표시 - 배치 - 사이즈 - 폰트 - 기타 - 배경 - 상태 순서로 기술한다.

순서 속성 의미 비고
1 display 표시
2 float 흐름
3 position 위치
4 width & height 크기
5 margin & padding 간격
6 border 외곽선
7 font 글꼴 축약형은 쓰지않으며, color, font-weight, font-size, line-height, font-family 순으로 기입
8 etc 기타 text-align, vertical-align, text-indent, white-space, text-decoration 등
9 background 배경
10 overflow 넘침
11 z-index 정렬

기타 규칙

주석

주석 기호( /*, */ )와 내용 사이에는 반드시 공백 한 칸이 있어야 한다.

추가 및 수정이 되어 주석을 기입하는 경우 해당 수정날짜로 시작하고 두 줄이상일 경우 범위로 표시하며, 한 줄일 경우 해당 선언 마지막 뒤에 표시한다.
