필수 브라우저

Internet Explorer 8.0(기준), IE9, IE10, IE11 , Chrome 최신버전 , Firefox 최신버전


옵션 브라우저

Internet Explorer 7, Safari 브라우저는 레이아웃이 크게 깨지지 않는 선까지 수용한다.


IE 호환모드 설정

인터넷 익스플로러는 <meta> 태그를 통해 어떤 버전의 인터넷 익스플로러를 사용해야 하는지를 명시하여 문서의 호환성을 지원한다.
edge mode를 사용하여 IE가 항상 최신 버전의 모드로 문서를 렌더링하도록 지정한다.
