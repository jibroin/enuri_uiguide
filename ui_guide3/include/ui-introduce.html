개요 및 목적

UI 가이드란?

내부 업무시스템 구축 시 주요 설계/디자인 과정에서 준수되어야 할 사용자 중심의 UI 원칙과 적용 예시를 명시한 가이드

목적

내부 작업자 간에 최소한의 준수사항을 정해 향후 개발 시에 일관성 있는 코드를 적용할 수 있게 하는 데 목적을 둔다.
일관성 있는 가이드를 제공하여, 내부 작업 자들간에 의사소통을 위한 도구로 사용되길 희망한다.
본 가이드를 바탕으로 상호 간에 근거를 제시할 수도 있으며, 이를 통해 의사소통 손실을 줄이고 시간을 효율적으로 사용할 수 있도록 한다.

웹표준, 웹접근성 및 웹 기술에 대한 이해를 바탕으로 업무에 임할 수 있게 본 가이드를 제시
